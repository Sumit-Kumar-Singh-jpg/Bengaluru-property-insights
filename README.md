# Bengaluru Property Insights

A web application and related scripts to extract, analyze, and visualize land and property data for Bengaluru.

## Architecture
- **bengaluru-insights/**: Next.js frontend application with a Tailwind CSS / Stitch design system and Leaflet map integration.
- Data Extraction and Parsing Scripts: JS scripts to scrape or process property data (`extract_data.js`, `fetch_html.js`, etc.).
- SQL Database details: `updated_bengaluru_lands_data.sql` contains the schema and parsed data.

## Getting Started
Navigate to the `bengaluru-insights` directory, install dependencies using `npm install`, and run the development server via `npm run dev`.
